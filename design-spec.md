# Aoki Estate — デザイン仕様書（design-spec.md）

---

## 1. プロジェクト概要

- **サイト名**: Aoki Estate（アオキ・エステート）
- **業態**: 架空の地域密着型不動産会社（賃貸+売買の両方を取扱）
- **想定エリア**: 東京都杉並区・中野区・世田谷区を中心とした城西エリア
- **目的**: ポートフォリオ掲載用の架空クライアントワーク。物件を探すエンドユーザーにとって使いやすく、かつ案件依頼者（不動産会社オーナー）が「このクオリティで自社サイトを作ってほしい」と感じるサイト。
- **デザインコンセプト**: モダン×ナチュラル。アースカラーを基調に、大胆な明朝体タイポグラフィで個性を出す。落ち着きと信頼感を保ちつつ、「ただの不動産サイト」に収まらない洗練さを追求。
- **制作信条**: 「差別化と個性」「AI作成感の排除」「タイポグラフィへのこだわり」を最重視。

## 3. タイポグラフィ

### フォントファミリー
```
--font-heading-ja:  "Noto Serif JP", "Yu Mincho", "Hiragino Mincho ProN", serif
--font-heading-en:  "Cormorant Garamond", "Playfair Display", Georgia, serif
--font-body:        "Noto Sans JP", "Hiragino Kaku Gothic ProN", sans-serif
--font-ui:          "Poppins", "Noto Sans JP", sans-serif
```

### 使い分けルール
- **ページタイトル・キャッチコピー・セクション見出し（日本語）**: --font-heading-ja（明朝体）
- **セクション見出し（英語ラベル）**: --font-heading-en（セリフ体）
- **本文・説明文・フォーム**: --font-body（ゴシック体）
- **ボタン・タグ・バッジ・UI要素**: --font-ui（Poppins系）

### サイズスケール
| 要素 | サイズ | フォント | 備考 |
|------|--------|----------|------|
| Hero キャッチコピー | clamp(40px, 7vw, 72px) | heading-ja | 太字700、letter-spacing: 0.05em |
| ページタイトル | clamp(28px, 4vw, 48px) | heading-ja | 太字700 |

## 4. レイアウト方針

- **最大幅**: 1200px（コンテナ）
- **セクション間余白**: 100px〜140px（PC）、64px〜80px（モバイル）
- **カード角丸**: 8px（控えめ。モダンナチュラルは丸すぎない方が合う）
- **ボタン角丸**: 4px（角ばりに近い。不動産の信頼感）
- **シャドウ**: `0 2px 16px rgba(44,44,44,0.06)`（控えめ）
- **背景パターン**: セクション交互に --color-bg と --color-bg-alt。装飾は最小限。
- **余白の哲学**: 「詰めるより、空ける」。要素間の余白を広くとり、情報の密度を下げることで高級感と可読性を両立。

### レスポンシブブレークポイント
```
--bp-sm:  640px
--bp-md:  768px
--bp-lg:  1024px
--bp-xl:  1280px
```

## 5. 共通コンポーネント仕様

### 5.1 ヘッダー / グローバルナビゲーション
- 固定ヘッダー（position: fixed）
- 背景: 初期は透過、スクロール後に --color-surface + border-bottom + 軽い shadow
- 左: ロゴ（「Aoki Estate」セリフ体テキストロゴ + 小さなアイコン）
- 右: ナビリンク — 物件を探す（ドロップダウン: 賃貸/売買）/ 会社案内 / スタッフ / オーナー様へ / コラム / 採用情報
- 右端: 「お問い合わせ」CTAボタン
- モバイル（768px以下）: ハンバーガー → ドロワーメニュー
- ナビリンクのフォント: --font-ui、14px、letter-spacing: 0.04em

### 5.2 フッター
- 背景: --color-footer-bg
- 上部に 1px の --color-accent ライン
- 3カラム構成:
  - 左: ロゴ + 会社概要テキスト2行
  - 中: ナビリンク（白文字）
  - 右: 営業時間 + 連絡先 + SNSアイコン
- 最下部: コピーライト `© 2025 Aoki Estate Inc. All Rights Reserved.`

### 5.3 ボタン
- **Primary**: --color-accent 背景、白文字、角丸4px、ホバーで --color-accent-dark
- **Secondary**: 透明背景、--color-accent ボーダー + テキスト、ホバーでfill
- **Ghost**: 透明背景、白ボーダー + 白テキスト（ダーク背景上）
- **Text Link**: --color-primary テキスト + 下線、ホバーで色濃く
- すべてのボタンに transition: 0.25s ease

### 5.4 セクション見出しパターン
```
英語ラベル（小、uppercase、--color-text-muted、Cormorant Garamond 400、letter-spacing: 0.2em）
日本語タイトル（大、明朝体 700、--color-heading）
```
- 装飾ラインは使わない（タイポの力と余白で成立させる）
- 左寄せが基本（中央寄せはHeroとCTAセクションのみ）

### 5.5 写真の扱い
- すべての物件写真・イメージ写真に統一的なCSS filter を適用:
  `filter: brightness(1.02) saturate(0.92) contrast(1.02);`
  → 若干明るく、彩度を微妙に落とすことで、暖かみのあるトーンを統一
- ホバー時: `filter: brightness(1.06);` で微かに明るく

### 5.6 ページ遷移共通構造
- すべての下層ページは共通の header と footer
- ページ上部にページヒーロー帯: 高さ180〜240px、--color-bg-warm 背景
  - 英語ラベル（小）+ 日本語ページタイトル（明朝体・大）
  - パンくずリスト: TOP > ページ名（直下に配置）

  - ハンバーガーメニュー + ドロワー
  - 物件検索フィルタリング（サイドバー条件 → カードのフィルタリング、data属性で管理）

  - ヘッダースクロール時スタイル変化

## 9. 画像方針

本プロジェクトでは、すべてのビジュアル箇所に**実画像を使用する**方針とする。画像はあおきさんがChatGPT（DALL-E）等で生成・用意する。

### 実装時の方針
- 実装段階では、すべての画像箇所に**プレースホルダー（CSSグラデーション + テキストラベル）**を配置する。
- プレースホルダーには、後日差し替え時にわかりやすいよう、ファイル名・推奨サイズ・用途をHTMLコメントまたは alt 属性で明示すること。
- サイト仮完成後、あおきさんがChatGPTに投げるための**画像生成プロンプト・サイズ・比率の一覧表**を別途出力する予定。Claude Code は仮完成時にその一覧作成用の情報（各画像の用途・推奨サイズ・雰囲気のキーワード）をまとめて報告すること。

### 画像が必要な箇所の一覧

**トップページ:**
- Hero背景画像（hero.jpg）— 温かみのある街並み or 窓からの光
- エリアガイド画像 ×3（area-suginami.jpg, area-nakano.jpg, area-setagaya.jpg）

**物件写真:**
- 各物件メイン写真 ×6件（property-[slug]-main.jpg）
- 各物件サブ写真 ×4枚ずつ ×6件 = 24枚（property-[slug]-sub01〜04.jpg）

**会社概要:**
- 代表写真（staff-representative.jpg）
- 事務所外観 or 内観（office.jpg）

**スタッフ紹介:**
- スタッフ写真 ×4名（staff-01〜04.jpg）

**コラム:**
- コラムサムネイル ×5件（column-01〜05.jpg）

**その他:**
- お客様の声の背景・装飾画像（必要に応じて）
- オーナー向けセクションのイメージ画像（owner-image.jpg）

### 写真の統一トーン
すべての物件写真・イメージ写真に以下のCSS filterを適用し、サイト全体のトーンを統一:
```
filter: brightness(1.02) saturate(0.92) contrast(1.02);
```
